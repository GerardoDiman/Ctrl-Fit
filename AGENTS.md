# Agent Knowledge Base & Guidelines

¡Bienvenido al proyecto! Este archivo sirve como índice principal del conocimiento persistente, reglas y habilidades de la IA para este proyecto (Ctrl + Fit).

## Instrucciones para el Asistente AI
1. **Siempre consulta el directorio `.agents`** para reglas específicas y habilidades antes de comenzar cualquier tarea importante, especialmente al tomar decisiones arquitectónicas o escribir nuevas funcionalidades.
2. **Idioma Principal**: Toda la comunicación, así como la redacción de planes de implementación, tareas (tasks), tutoriales (walkthroughs) y documentación, **DEBE ser estrictamente en español**.
3. **Actualiza la base de conocimiento**: Cuando adoptemos nuevas convenciones, o cuando notes que necesitas recordar instrucciones específicas para este proyecto a futuro, estás autorizado y **obligado** a actualizar los archivos dentro de `.agents/rules` y `.agents/skills`.
4. No dependas únicamente de tu conocimiento general si existe una regla específica en la carpeta `.agents`.

## Estructura de Directorios
- `.agents/rules/`: Contiene pautas estrictas sobre el uso del stack tecnológico, formateo y restricciones del proyecto.
- `.agents/skills/`: Contiene implementaciones preferidas, flujos de trabajo (UI, Backend, Testing, Deploy) y habilidades específicas que debes aplicar.

Siéntete libre de leer y/o modificar el contenido de estos directorios para asegurar que tu contexto esté siempre alineado con la evolución del proyecto.
